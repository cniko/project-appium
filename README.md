### 下载安装

* 需要环境
```shell
JDK
安卓环境
appium-server
uiautomatorviewer
```

* 下载源码:

```shell
git clone https://github.com/spider-xyz/proxy-spider.git
```

* 安装依赖:

```shell
pip3 install -r requirements.txt
python3 appium_chelun.py
python3 appium_douyin.py
python3 appium_taobao.py
```
